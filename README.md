# ai-agent-practice

AI Agent Engineering 실습을 위한 기본 프로젝트 저장소입니다.

## 프로젝트 구조

```text
ai-agent-practice/
├── README.md
├── .env.example
├── requirements.txt
├── app/
├── agents/
├── tools/
├── prompts/
├── data/
├── outputs/
├── notebooks/
└── docs/
```

## 폴더 역할

| 폴더 | 역할 |
| --- | --- |
| `app/` | 실행 진입점과 API 또는 웹앱 코드를 저장합니다. |
| `agents/` | 역할별 에이전트 로직을 저장합니다. |
| `tools/` | 검색, 계산, 파일 처리 등 에이전트가 사용할 도구를 저장합니다. |
| `prompts/` | 시스템 프롬프트와 작업 프롬프트를 저장합니다. |
| `data/` | 실습용 문서, CSV, PDF 등 입력 데이터를 저장합니다. |
| `outputs/` | 실행 결과, 로그, 생성 파일을 저장합니다. |
| `notebooks/` | 실험용 Jupyter Notebook을 저장합니다. |
| `docs/` | 프로젝트 설명, 설계 메모, 사용법 문서를 저장합니다. |
