# 회사 업무 정책 샘플

## 휴가 정책

연차 휴가는 사전 승인 후 사용할 수 있습니다.
긴급한 경우에는 사후 승인 절차를 따를 수 있습니다.
연차 신청 시에는 업무 대체자와 진행 중인 업무 인수인계 내용을 함께 작성하는 것이 좋습니다.

## 재택근무 정책

재택근무는 팀장 승인 후 사용할 수 있습니다.
재택근무일에는 업무 시작 전 당일 주요 업무 계획을 공유하고, 업무 종료 전 진행 결과를 정리합니다.

## 문서 보안 정책

외부 공유 문서에는 민감정보, API Key, 고객 개인정보를 포함하지 않습니다.
공개 가능한 문서와 내부 전용 문서를 구분해 관리해야 합니다.
